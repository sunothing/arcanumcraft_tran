Ректификационная колонна
Экологически недружелюбная
<&distillationtower0>Ректификационная колонна - крупная многоблочная структура, разделяющая сырую нефть на фракции.<np>
Она строится, как показано на схеме и формируется ударом молота инженера по редстоуновому инженерному блоку.

Подача сигнала редстоуна на панель управления машины, будет останавливать работу. Это поведение можно инвертировать с помощью Отвёртки.

Сырая нефть может быть загружена в ректификационную колонну через задний люк, помеченный синей точкой, где она будет нагреваться до тех пор, пока не распадётся на фракции. Производит дизельное топливо и имеет шанс производства битума, как побочного продукта.

Ректификационная колонна использует <config;i;distillationtower_operationcost> Flux за операцию, перегоняя 75 mB сырой нефти в дизельное топливо, бензин, смазку и битум за тик. Полученную жидкость можно вывести сбоку из люка, помеченного оранжевой точкой; битум же выводится спереди.<np>

Дизельное топливо используется для <link;diesel_generator;§o§nпроизводства §o§nэнергии§r>, смазка для <link;immersivepetroleum:lubricant;§o§nсмазывания §o§nмашин§r>, а бензин для <link;immersivepetroleum:portablegenerator;§o§nзапитки §o§nдвигателей§r>, <link;immersivepetroleum:speedboat;§o§nлодок§r>.
Битум используется для создания <link;immersivepetroleum:asphalt;§o§nАсфальтобетона§r> и может быть переработан далее для получения нефтяного кокса в <link;immersivepetroleum:cokerunit;§o§nУстановке для коксования§r>.<np>
